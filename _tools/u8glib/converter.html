Bitmap Converter

This tool converts a JPEG or PNG image file to a U8Glib-compatible C/C++ array that can be used in Marlin as a custom _Bootscreen.h file or as icons for Graphical Displays. Darker colors in the source image will produce "on" pixels, while lighter colors will be converted to transparent pixels.

If this tool doesn't work in your browser, use the server-side tool at digole.com.
